Berlin subway distance project

Visit online

The Berlin subway distance project comprised two parts.

First, I wrote python to analyze the distance from each property in the Berlin Hauskoordinaten data set to its nearest subway entrance.

After running that python script in QGIS, I uploaded the resulting data to a database, using CodeIgniter/PHP/MySQL. This stage included a simple pagination routine, along with some basic mapping of property locations.

Role
Python and PHP development, basic front-end design
